# QuickEats - Food Delivery App 🍔

تطبيق أندرويد متكامل لطلب الطعام، تم بناؤه كشروع جامعي باستخدام لغة **Java** و **Android Studio**. يتميز التطبيق بتصميم عصري وتجربة مستخدم سلسة تتكون من 10 واجهات برمجية.

## ✨ مميزات التطبيق
* **نظام مستخدم متكامل:** واجهات تسجيل الدخول (Login) وإنشاء حساب جديد (Sign Up).
* **تصفح الأطعمة:** عرض قائمة الوجبات باستخدام `GridView` و `Adapters` مخصصة.
* **تفاصيل الوجبات:** واجهة خاصة لكل وجبة لعرض المكونات والسعر.
* **سلة المشتريات (Cart):** إمكانية إضافة الوجبات، حساب السعر الإجمالي، وحذف العناصر.
* **إتمام الطلب:** واجهة نجاح الطلب (Order Success) مع رسوم توضيحية.
* **قائمة جانبية (Navigation Drawer):** للتنقل السريع بين أقسام التطبيق المختلفة.

## 📱 واجهات التطبيق (10 واجهات)
1. **Splash Screen:** واجهة ترحيبية عند فتح التطبيق.
2. **Onboarding Screen:** مقدمة تعريفية للتطبيق.
3. **Login Activity:** لتسجيل دخول المستخدم.
4. **Sign Up Activity:** لإنشاء حساب جديد.
5. **Home Activity:** الواجهة الرئيسية لعرض أصناف الطعام.
6. **Food Details Activity:** تفاصيل الوجبة المختارة.
7. **Cart Activity:** إدارة الطلبات وحساب السعر النهائي.
8. **Orders Activity:** سجل الطلبات السابقة.
9. **Profile Activity:** معلومات المستخدم.
10. **Order Success Activity:** تأكيد إتمام عملية الشراء بنجاح.

## 🛠 التكنولوجيا المستخدمة
* **Language:** Java
* **UI Layouts:** XML (Material Design)
* **Storage:** Static Lists / Intent Extras
* **Tools:** Android SDK, Vector Assets, AppCompat Libraries
